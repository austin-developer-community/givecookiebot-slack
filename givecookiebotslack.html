givecookiebotslack.bot.handle_message(event_data: dict)[source]¶

Message handler

Parses message for cookie emoji (:cookie:) and a user mention, then changes mentioned user’s status to include cookie emoji for 3 hours.

Returns

True if successful, otherwise the error code given by the Slack API.

givecookiebotslack.tokens module¶

Get User and Bot tokens.

Uses Slack’s OAuth 2.0 to fetch the User and Bot tokens for interfacing with Slack’s API.

Module contents¶

givecookiebot-slack: Slack bot that adds a cookie emoji (🍪) to a user’s status for 3 hours when that user is ‘given’ a cookie in chat.
